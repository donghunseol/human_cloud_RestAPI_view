# 프로젝트 소개
`SHINE`은 **누구나 사용 가능한 취업 페이지**입니다.</br>
다양한 공고를 통해 지원을 하고 원하는 인재를 찾기 위한 공고를 올리는게 가능한 페이지입니다.</br>

# 프로젝트 목표
- Model을 ResponseEntity 응답
- ResponseEntity ORM
- JSON으로 변환
- image base64로 변환

## 브랜치 전략

- 브랜치 명은 각 인원별 `패키지/기능/이니셜`로 컨벤션을 맞춰 진행
